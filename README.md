# dailyTasksWebApp

With this tool you can manage your daily tasks.

Steps:
	1- Create a new task
	2- Sort in your favourite order
	3- when finish, delete the task
	
Tools used:
	- HTML
	- CSS
	- BootsWatch
	- JavaScript
	
NOTE: if you close or restart the page, you will lost your data. I'm working on adding a database.
